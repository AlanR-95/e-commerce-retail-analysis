# Aplicando el análisis de datos en la industria del Retail (e-commerce)

![mejorar-la-experiencia-post-venta](https://user-images.githubusercontent.com/69488042/90622402-6cf85680-e1eb-11ea-991d-12c23fde0bce.jpg)

A partir de la información de las ventas a lo largo de un año de una empresa británica de e-commerce se realizaron las siguientes actividades:

* Segmentación de clientes: Por medio del método RFM se logró identificar y agrupar los clientes mas valiosos para la empresa.

* Valor del cliente: Por medio del método CLV (Customer Lifetime Value) se procedió a modelar las probabilidades de que un cliente particular vuelva a comprar en un futuro, cúando y cúanto. Se utilizó la libreria "lifetimes" de Shopify para llevar a cabo el análisis.

* Cesta de Compras: Se analizaron las relaciones entre los items comprados por los clientes y se establecieron reglas con porcentajes de probabilidad que indican qué artículos se suelen comprar al mismo tiempo, permitiendo conocer más sobre los hábitos de consumo de los clientes habituales. Se utilizó la librería mlxtend para poder crear las reglas de asociación.

Todos los datos fueron limpiados y procesados según se requería para cada actividad.
